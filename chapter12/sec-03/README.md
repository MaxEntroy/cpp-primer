2. 对于单元测试，可以通过增加成员的get接口来避免我在demo-02当中的实现，我承认我实现的不好，但是private method是没有了

我自己的实现，整体来说，我认为思路还是比较清晰的，对于问题的分析还算透彻，接口语义也比较清新.不足的地方在于，单元测试消耗了我很多时间，
